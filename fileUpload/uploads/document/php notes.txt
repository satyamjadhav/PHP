TYPES OF SUPERGLOBAL VARIABLES:

1.GET METHOD
2.POST METHOD
3.REQUEST METHOD  - WHICH METHOD IS TO BE USE FOR SENDING THE INFORMATION:
4.COOKIE - A BROWSER MEMORY
5.SESSION - A SERVER MEMORY
6.$_ENV - INFO ABOUT THE SERVER ENVORNMENT
7.$_SERVER- TO GET THE IP ADDRESS OF THE CLIENT
8.$GLOBALS -TO CREATE THE GLOBAL VARIABLE
9.$_FILES -TO DEALS WITH FILES
